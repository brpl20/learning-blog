# 3.14 Cortes
- Razor Tool : (C)
- Ctrl + K => Cortar Direto 
- Q => Corta e Trás
- W => Corta e Avança 
	* Soft And Tail Editing 
- Sequence => Close Gap 
- Também da pra cortar diminuindo começo e fim (TRIM)
- Ripple Editing Tool => Empurra
- Rooling = Dimuni
- Perguntas?
	* Como pular de um corte ao outro? 
	* Como dar o play mais rápido?
	* Como selecionar cortes pequenos?
- Jump-Cuts: Corte de pulo, quando tirar pausa
- J-Cut: Deixa uma cena e começa audio
- L-Cut: Deixa áudio e começa uma cena
- Cut In Action: Fechando notebook que vira um livro (encontrar o frame correto) - precisa de ajuda na gravação 
- Match-Cut: Relógio na parede e depois relógio no pulso (Mantem a "coisa" principal centralizada) - precisa de ajuda na gravação

# 3.15 Advanced Trim Mode 
- Dois cliques entre os cortes
- Ctrl + Setinha => Vai trinando
- Ctrl + Shif Setinha => Vai trinando 5 + 5
- Fácil para adicionar transição suave (clicar na sombrinha)
- É possível customizar para aumentar ou diminuir os trims nas configurações
- 24 frames => 1 segundo
